# Stable Diffusion

## 機能

- ローカル環境、Google Colab 環境それぞれの Notebook を提供
  - txt2img, img2img に対応
  - prompt_correction パッチ ([参考](https://zenn.dev/td2sk/articles/eb772103a3a8ff)) を実装済み
